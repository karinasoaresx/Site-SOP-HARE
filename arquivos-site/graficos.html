Criação de Gráficos

Gráficos são representações visuais para demonstrar dados estatísticos, assim facilitando a compreensão, além de valorizar o trabalho.

Aqui traremos a criação de diferentes gráficos usando o excel.

Passo 1

Navegue até o menu "Inserir".

Passo 2

Selecione o tipo de gráfico.

Passo 3

Agora é hora de criar o gráfico.

Coluna agrupada

Um gráfico de colunas agrupadas é usado para comparar valores entre algumas categorias.

Barras agrupadas

Um gráfico de barras agrupadas é usado quando o texto da categoria é extenso demais.

Linhas

Um gráfico de linhas é usado para exibir as tendências, dando valor à ordem de dados.

Dispersão

Um gráfico de dispersão é usado para comparar pelo menos dois conjuntos de valores ou dados.

Pizza

Um gráfico de pizza é usado para mostrar as proporções de um todo, ou seja, números que sejam %.

Combinação

Um gráfico de combinação é usado quando para comparar valores e quando a ordem não for crucial.
